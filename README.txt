Simple python Tic Tac Toe project. 

Allows the user to input if it will be 1 player or 2 players.

If 1 player:
- keeps track of the turn.
- If user's turn they input the next move.
- If computer's turn, randomly selects next move from available moves.

If 2 players:
- keeps track of the turn.
- Takes input of next move selection from each user.

Validates for legal moves. Validates for Tie or Game win.
